
INSTALLATION:

Simply extract the zip file to a local directory.


USAGE:

Run the unzipped bat file in the toplevel directory.
Check that the properties are correcly set:
	- Path to DVT
	- Path to destination
	
Make sure that the account under which the tool is run, has write access to the destination location.

Browse to the path where the DICOMDIR or single is placed. 
Fill in the description and choose on of the options 'clinical' or 'non-clinical'. 
Then press submit and the tool will execute DVT and upload the images to the correct destination.
